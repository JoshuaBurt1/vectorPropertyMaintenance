# Vector Property Maintenance

Landing page for Vector Property Maintenance. 

Monorepo architechture: 
* server reads and writes to Firestore (NoSQL) database. Hosted on Render.
* web (front-end) communicates through server only. Hosted on Firebase. 
* mobile application is a WebView of the front-end. 

## Getting Started

server: https://vectorpropertymaintenance.onrender.com/
front-end: https://vectorpm-df058.web.app
